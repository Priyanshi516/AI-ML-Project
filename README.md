# 🍃 Multimodal Diagnostic Assistant

An intelligent medical diagnostic assistant built using Streamlit and Generative AI (Gemini Vision API) to analyze multimodal medical data with a calming, user-friendly interface.

## ✨ Features
- **Multimodal Analysis:** Upload medical images/reports and get instant AI-driven insights.
- **Calming Nature UI:** Minimalist, user-focused design based on soft green leaf themes for a relaxed workflow.
- **Project Tracking:** Real-time diagnostics workspace for medical project evaluation.

## 🛠️ Tech Stack
- **Frontend/UI:** Streamlit (Python)
- **AI Backend:** Google Gemini API (Generative AI & Vision)
- **Language:** Python 3.10+
